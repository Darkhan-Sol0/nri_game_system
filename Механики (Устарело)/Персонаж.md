## Общая схема

| Шаг           | Что делаем                                   |
| ------------- | -------------------------------------------- |
| Раса          | Выбираем чистую расу или собираем полукровку |
| Предыстория   | Кем был до приключений                       |
| Источник силы | Откуда берётся магия                         |
| Путь развития | В каком направлении развиваемся              |
| Навыки        | Распределяем стартовые навыки                |
| Снаряжение    | Что есть с собой                             |
| Имя и детали  | Как зовут, как выглядит, характер            |
|               |                                              |
|               |                                              |
|               |                                              |

## Части тело

| Часть тела  | Одежда        | Оружие                                            |
| ----------- | ------------- | ------------------------------------------------- |
| Голова      | Головной убор | Нет (Может быть рогатый шлем)                     |
| Туловище    | Одежда        | Нет (Может быть Шипастая броня)                   |
| Левая Рука  | Перчатка      | Оружие/Посох/Двуручник/Щит/Боевая перчатка/Костет |
| Правая Рука | Перчатка      | Оружие/Посох/Двуручник/Щит/Боевая перчатка/Костет |
| Левая Нога  | Обувь         | Нет (Может быть Стальной сапок)                   |
| Правая Нога | Обувь         | Нет (Может быть Стальной сапок)                   |
## Шаг 1. Раса

Игрок выбирает расу. [[Расы]]

## Шаг 2. Предыстория

Игрок выбирает предысторию. [[Предыстория]]
Можно взять готовое, либо придумать самому.
- влияет на отыгрышь, возможны бонус для проверки предыстории и т.д. все на усмотрение ГМа.
## Шаг 3. Источник силы

Игрок выбирает источник силы. [[Источник Силы]]
Можно взять готовое, либо придумать самому используя шаблон.
- основной источник магической силы игрока.

## Шаг 4. Профессия (необязательно)

Игрок выбирает профессию. [[Специализация (Устарело)]]
Можно взять готовое, либо придумать самому используя шаблон.
- влияет на отыгрышь
- помогает для новичков в развитие персонажа во время прокачки.
- если игрок придумал свое, то влияние только для отыгрыша.
## Шаг 5. Характеристики

Игрок распределяет характеристики. [[Характеристики]]

## Шаг 6. Снаряжение

Игрок выбирает стартовые снаряжения. [[Снаряжения]]
- стартовый набор предметов.

## Шаг 7. Имя и детали

- Имя
- Пол
- Возраст
- Внешность
- и т.д.

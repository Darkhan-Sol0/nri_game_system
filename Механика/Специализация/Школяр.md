# Школяр
## Требование

Нужно: Иноватор

## Описание

## Эффект

+1 Алхимии, +1 Магическое Знания, +1 Механик

Получает способность: _Патент_

Нужны инструменты: Можно создавать предметы из _Рецепты Школяра_

_Патент_: Дает _Кубик Бонус_ для использование самосозданных вещей. Увеличивает стоимость продажи самосозданных вещей на 50%.

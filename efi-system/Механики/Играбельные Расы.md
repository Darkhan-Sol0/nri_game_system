# Полукровки:
Игрок выбирает двух родителей из списка чистых рас.
### Планарные полукровки:
- Тифлинг = чистая раса + Демоническая Кровь (не игровая)
- Серафим = чистая раса + Ангельская Кровь (не игровая)

# Уровень 1

- Духа Огня

## Тотем: Духа Огня (Орел):
Получите предмет Силы: Тотем Духа Огня
Получает Способности:
- Призыв: Огненного Орла
- Тотем Силы Огня

| Название        | ОВ                 | Эффект                                                                |
| --------------- | ------------------ | --------------------------------------------------------------------- |
| Призыв          | 2 ОВ               | Призывает Тотемного Духа до конца боя                                 |
| Огненный Орел   | Существо           | Очки здоровья: 3<br>Порог урона: 3-6-9<br>Атака: 1к6+Мощь Урона Огнем |
| Тотем Силы Огня | 1 ОВ, Концентрация | Дает всем союзникам в радиусе Средней Дистанции +1к4 урона Огнем      |

# Уровень 1
Получает Способности:
- Сверх Реакция
- Шквал Ударов
Улучшает:
- Рукопашный Урон - 1к4+Сила->1к6+Сила

| Название      | ОВ       | Эффект                                           |
| ------------- | -------- | ------------------------------------------------ |
| Сверх Реакция | Пассивно | Без Брони: +Ловкость к уклонению                 |
| Шквал Ударов  | 2 ОВ     | Провести две Рукопашные Атаки за 1 Очко Действия |
# Уровень 2
Выбрать Стиль:
- Стиль Пьяного Кулака
- Стиль Кулака Ярости
- Стиль Слепого Кулака
- Стиль Четырех Стихий
## Стиль: Пьяный Кулак
Получает Способности:
- Пьяная Реакция
- Толерантность к Алкоголю
Меняет: 
В состоянии Опьянения:
- Сверх Реакция -> Пьяная Реакция
- Рукопашный Урон - 1к6+Сила->1к10+Сила
Без состоянии Опьянения:
- Пьяная Реакция -> Сверх Реакция
- Рукопашный Урон - 1к10+Сила->1к6+Сила

| Название                 | ОВ       | Эффект                                                                                                                              |
| ------------------------ | -------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Пьяная Реакция           | Пассивно | Без Брони: +(Ловкость)к4 к уклонению                                                                                                |
| Толерантность к Алкоголю | Пассивно | Для получения Состояния Опьянение нужно x2 Алкоголя.<br>Опьянение длится до конца боя.<br>Все Штрафы Опьянения становятся Бонусами. |

## Стиль: Кулак Ярости
Нужно: Первобытная Ярость
Получает Способности:
- Умиротворение
Меняет:
- Ярость -> Хладнокровие

| Название      | ОВ        | Эффект                                                                                                                                               |
| ------------- | --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Хладнокровие  | Пассивное | +1 к Ярости, за каждое попадание по Цели.<br>-1 к Ярости, за каждое попадание по Вам.                                                                |
| Умиротворение | 3 ОВ      | Потратить всю Ярость. Наносит Сильный удар по цели, +1к4 за каждую потраченную ячейку Ярости, и отталкивает цель на расстояние 5+Сила+Ярость метров. |

## Стиль: Слепой Кулак
Нужно: Необычное Зрение: Слепота
Получает Способности:
- Эхолокационный Рывок
Улучшает:
- Эхолокация -> Концентрационная Эхолокация 

| Название                    | ОВ                     | Эффект                                                                                                                                                                                                                                                                                                                 |
| --------------------------- | ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Концентрационная Эхолокация | Пассивно, Концентрация | Знаете местоположение всех живых в радиусе Средняя дистанция, и только движения в радиусе Дальняя дистанция.<br>Концентрация: Концентрируетесь на цели в радиусе Средняя Дистанция. +2 к Уклонения от Атак и Заклинаний цели на которую Концентрируетесь. +2 к Атаке и Заклинаний по цели на которую Концентрируетесь. |
| Эхолокационный Рывок        | 3 ОВ                   | Перемещаетесь за спину цели на которую концентрируетесь, и наносите АвтоКрит при попадании по Цели на которую Концентрируетесь.                                                                                                                                                                                        |

## Стиль: Кулак Четырех Стихий
Получает Способности:
- Поглощение Эфира
- Заряженные Кулаки
- Выплеск Эфира

| Название          | ОВ                  | Эффект                                                                                                                                     |
| ----------------- | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Поглощение Эфира  | На отдыхе           | Поглощает Эфир Стихии, для получения силы Стихии. Длительность до конца дня или долгого отдыха.                                            |
| Заряженные Кулаки | 2 ОВ - Концентрация | Ваши Атаки получает 1к4 дополнительного урона стихии поглощенного Эфира.                                                                   |
| Выплеск Эфира     | 2 ОВ                | ВСЕ в конусе на расстоянии Ближней Дистанции наносится урон 1к8+Мощь урона стихии поглощенного Эфира. Эффект поглощенного Эфира пропадает. |

# Уровень 3
Получает Способности:
- Пять Сторон Света
- Сверх Скорость
- Перехват

| Название          | ОВ            | Эффект                                                                                                               |
| ----------------- | ------------- | -------------------------------------------------------------------------------------------------------------------- |
| Пять Сторон Света | 5 ОВ          | Нанеси 5 Рукопашных Атак со всеми усилениями разным целям в радиусе Средней Дистанции. Нельзя бить одну цель подряд. |
| Сверх Скорость    | Пассивно      | Увеличивает расстояние перемещения в 2 раза                                                                          |
| Перехват          | 2 ОВ, Реакция | Когда вас Атакуют, вы не получите урона и нанесете урон Атакующему равный его Урону.                                 |

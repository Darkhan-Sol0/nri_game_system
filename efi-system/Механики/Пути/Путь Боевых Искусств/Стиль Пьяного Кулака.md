# Уровень 1
Получает:
- Сверх Реакция
- Шквал Ударов
- Пьяная Реакция
- Толерантность к Алкоголю
Улучшает:
- Рукопашный Урон - 1к4+Сила->1к6+Сила
Меняет:
В состоянии Опьянения:
- Сверх Реакция -> Пьяная Реакция
- Рукопашный Урон - 1к6+Сила->1к10+Сила
Без состоянии Опьянения:
- Пьяная Реакция -> Сверх Реакция
- Рукопашный Урон - 1к10+Сила->1к6+Сила

| Название                 | ОВ       | Эффект                                                                                                                              |
| ------------------------ | -------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Сверх Реакция            | Пассивно | Без Брони: +Ловкость к уклонению                                                                                                    |
| Шквал Ударов             | 2 ОВ     | Провести две Рукопашные Атаки за 1 Очко Действия                                                                                    |
| Пьяная Реакция           | Пассивно | Без Брони: +(Ловкость)к4 к уклонению                                                                                                |
| Толерантность к Алкоголю | Пассивно | Для получения Состояния Опьянение нужно x2 Алкоголя.<br>Опьянение длится до конца боя.<br>Все Штрафы Опьянения становятся Бонусами. |

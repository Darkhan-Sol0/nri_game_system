# Кубики

| Кубик | Что? |
| --- | --- |
| к2 | Монетка |
| к4 | Четырех гранная кость (Пирамидка) |
| к6 | Шести гранная кость (Кубик) |
| к8 | Восьми гранная кость (Октаэдр) |
| к10 | Десяти гранная кость (Юла) |
| к12 | Двенацати гранная кость (Типо шар, но угловаты) |
| к20 | Двадцати гранная кость (Основная кость) |
| к100 | Либо два Десяти гранных кости (Юла), либо шар со значениями 1 - 100. |
# Объяснение

## Преимущество
- Преимущество - Бросок двух к20 и берется лучший результат.
- Двойное/Тройное и т.д. это количество дополнительных к20.
## Помеха
- Помеха - Бросок двух к20 и берется худший результат.
- Двойное/Тройное и т.д. это количество дополнительных к20.
## Бонус
- Бонус -N - Дает бонус равный N.
- Кубик Бонуса дает +1к4 к проверке.
- Двойное/Тройное и т.д. _Кубик Бонуса_, это количество дополнительных +1к4.
- Полный бонус - при проверке к20 есть только крит провал, все остальное успех, и крит успех.
## Штраф
- Штраф -N - Дает штраф равный N.
- Кубик Штрафа дает -1к4 к проверке.
- Двойное/Тройное и т.д. _Кубик Штрафа_, это количество дополнительных -1к4.
- Полный штраф - при проверке к20 есть только крит успех, все остальное провал, и крит провал.
## к20 Успехи и Неудачи
- Базовая сложность 10.
- Базовый порог крит успеха 20
- Базовый порог крит провала 1
- Уменьшение порога крит успеха = 20 - модификатор порога
- Увеличение порога крит провала = 1 + модификатор порога
- Изменение сложности = 10 + модификатор сложности или Значение по сложности
## Генератор сложности проверок

| Сложности | Модификатор | Значение |
| --- | --- | --- |
| Легчайшее | \-10 | 1 |
| Очень Легко | \-7 | 3 |
| Легко | \-5 | 5 |
| Обычно | 0 | 10 |
| Сложно | 5 | 15 |
| Очень Сложно | 7 | 17 |
| Невозможно | 20 | 20 |

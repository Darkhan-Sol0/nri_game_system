# Боевая Система

Система 3-х действий на ход + 1 свободное действие

| Действие   | 1 Основное                                                                             | 2 Основное                                                                             | 3 Основное                                                                             | Свободное                                                                   |
| ---------- | -------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| Что делать | Перемещение/Атака/Заклинание/Прием/Использовать Предмет/Концентрация/Удержание/Реакция | Перемещение/Атака/Заклинание/Прием/Использовать Предмет/Концентрация/Удержание/Реакция | Перемещение/Атака/Заклинание/Прием/Использовать Предмет/Концентрация/Удержание/Реакция | Все что не влияет на геймплей (Оскорбление, Неприличный жест, танец и т.д.) |
- Перемещение - передвижение на расстояние.
- Атака - Простая атака, простое заклинание.
- Заклинание - Заготовленные заклинания с использованием Очков Выносливости.
- Прием - Заготовленные приемы с использованием Очков Выносливости
- Использовать Предмет - Использование предметов (Свитки заклинаний, аптечки, зелье лечение и т.д.).
- Концентрация - Если было использовано заклинание/прием требующие концентрацию, то на начало хода, тратится столько очков действий, сколько требуется концентрации на поддержание заклинания/приема.
- Удержание - Если хочешь подготовить действие, для использование в не свой ход, то тратишь 1 очко действие на удержание.
- Реакция - Если в не свой ход, с помощью реакции совершил действие, тратишь 1 очко действия.

### Показатели

- Инициатива = к20 + ловкость
- Уклонение = 10 + ловкость + доп бонус уклонения
- Очки выносливости = 5 + телосложение + уровень персонажа
- Сопротивление физическим эффектам атаки/магии/приемам/заклинаниям = 10 + телосложение  + доп бонус сопротивление физики
- Сопротивление магическим эффектам атаки/магии/приемам/заклинаниям = 10 + воля  + доп бонус сопротивление магии
- Атака по части тела = 1к6 (1-лева нога 2-правая нога 3-левая рука 4-правая рука 5-тело 6-голова)
- Количество подготовленных заклинаний/приемов = 3 + интеллект + уровень персонажа
- Прицельная атака = увеличивает сложность попадание на 1
- Атака = 1к20 + доп бонус к атаке
- Магическая Атака = 1к20 + доп бонус к магической атаке
- Прием = 1к20 + доп бонус к приему
- Заклинание = 1к20 + доп бонус к заклинанию
- Успешная атака = Атака >= Уклонения
- Успешный прием = Прием >= Уклонения
- Успешная Магическая атака = Магическая атака >= Уклонения
- Успешное Заклинание = Заклинание >= Уклонения
- Урон физической атакой = кубик урона оружия (пример 1d6) + сила + доп урон атаки
- Урон магической атакой = кубик урона магической атаки (пример 1d6) + мощь + доп урон атаки
- Урон приема = кубик урона приема (пример 1d6) + сила + доп урон приему
- Урон заклинания = кубик урона заклинания (пример 1d6) + мощь + доп урон магии
- Крит урон физической атакой = двойной кубик урона оружия (пример 1d6 * 2) + сила
- Крит урон магической атакой = двойной кубик урона магической атаки (пример 1d6 * 2) + мощь
- Крит урон приема = двойной кубик урона приема (пример 1d6 * 2) + сила
- Крит урон заклинания = двойной кубик урона заклинания (пример 1d6 * 2) + мощь
- Показатель Защиты = Защита одетой брони на часть тела
- Фактический урон = Любой из выше перечисленных урона - Показатель Защиты Части Тела
- Успешность наложение физическим эффектам атаки/магии/приемам/заклинаниям = к20 + доп бонус эффекта атаки/магии/приемам/заклинаниям
- Успешность наложение магическим эффектам атаки/магии/приемам/заклинаниям = к20 + доп бонус эффекта атаки/магии/приемам/заклинаниям
- атаки/магии/приемам/заклинаниям наносят урон части тела, куда попал
- атаки/магии/приемам/заклинаниям по площади распределяет урон по всем частям тела (приоритет: Руки/Ноги(случайно или более здоровое) -> Тело -> Голова)

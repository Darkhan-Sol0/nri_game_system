# Объяснение
Одна ячейка, это +1 к проверки к20.
Для некоторых проверок к20, по желанию, можно добавить ***ПОДХОДЯЩИЙ*** модификатор Характеристик
***ПОДХОДЯЩИЙ*** - обсуждайте с ГМом, подходит ли в данной проверки к20, сила, мощь или внешность.

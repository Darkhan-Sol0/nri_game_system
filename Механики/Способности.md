# Объяснение

| Тег              | Значение                                 |
| ---------------- | ---------------------------------------- |
| Нужно            | Обязательное условие                     |
| Блок             | Блокирующее условие                      |
| Исключение       | Условия исключающее **Нужно** и **Блок** |
